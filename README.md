# 🎮 Prince of Light

A Unity game developed with C# that demonstrates gameplay programming, object-oriented design (OOP), and core game mechanics.

## 🛠 Tech Stack
- Unity
- C#
- Git & GitHub

## ✨ Features
- Character movement and controls
- Jump and attack mechanics
- Enemy interaction and collision system
- Scene and UI management

## 📂 Project Structure
- `Assets/` – Scripts, sprites, animations, and scenes
- `Packages/` – Unity package dependencies
- `ProjectSettings/` – Project configuration
